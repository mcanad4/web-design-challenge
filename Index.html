Summary: Latitude vs. X

Summary: The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this, we first pulled data from OpenWeatherMap API to assemble a data set on over 500 cities.

After assembling the dataset, we used Matplotliob to plt various aspects of the weather vs. latitude. Facotrs we looked at includeed: tempature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explamations and descriptions of any trends and correlations witnessed.
